# OrgChart Populator

## Configuration

Create a file called `private.json` as a sibling to `main.py` and populate it as follows:

```json
{
  "API_KEY": "",
  "BASE_ID": "",
  "TABLE_ID": "",
  "LDAP_HOST": "",
  "LDAP_OC": ""
}
```

## Installation

Clone and then [ `pipenv`](https://pipenv.pypa.io/en/latest/) install.

## Running

Configure as explained above, then look in `main.py`.
